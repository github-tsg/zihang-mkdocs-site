\# 设备报修流程



\## 一、报修前准备

\### 故障自查步骤

1\. \*\*基础检查\*\*

&nbsp;  - 检查电源连接是否正常

&nbsp;  - 确认设备指示灯状态

&nbsp;  - 测试不同电源插座



2\. \*\*简单排查\*\*

&nbsp;  - 重启设备尝试恢复

&nbsp;  - 检查连接线缆是否松动

&nbsp;  - 确认设备是否过热



3\. \*\*信息收集\*\*

&nbsp;  - 记录设备型号、序列号

&nbsp;  - 拍摄故障现象照片/视频

&nbsp;  - 记录错误代码或提示信息



\## 二、正式报修渠道

\### 在线报修系统

1\. \*\*访问地址\*\*：http://it-support.company.com

2\. \*\*登录账号\*\*：员工工号/企业邮箱

3\. \*\*填写内容\*\*：

&nbsp;  - 设备类型（电脑/打印机/网络设备等）

&nbsp;  - 故障描述（尽量详细）

&nbsp;  - 紧急程度（高/中/低）

&nbsp;  - 期望处理时间



\### 电话报修

\- \*\*紧急热线\*\*：分机 8001（7×24小时）

\- \*\*工作时间\*\*：分机 8002（9:00-18:00）

\- \*\*需提供信息\*\*：

&nbsp; - 姓名、部门、工号

&nbsp; - 设备位置（楼层、座位号）

&nbsp; - 故障现象简述
