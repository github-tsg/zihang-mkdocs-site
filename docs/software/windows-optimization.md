\# Windows 系统优化技巧



\## 系统性能优化

1\. \*\*禁用开机自启动程序\*\*

&nbsp;  - 按 `Ctrl + Shift + Esc` 打开任务管理器

&nbsp;  - 切换到"启动"选项卡

&nbsp;  - 禁用不必要的启动项（如聊天工具、云盘等）



2\. \*\*调整电源计划\*\*

&nbsp;  - 控制面板 → 电源选项

&nbsp;  - 选择"高性能"或创建自定义计划

&nbsp;  - 关闭硬盘休眠时间（设置从不）



3\. \*\*清理磁盘空间\*\*

&nbsp;  - 运行 `cleanmgr` 清理临时文件

&nbsp;  - 卸载不常用软件

&nbsp;  - 使用 `TreeSize` 分析大文件



\## 网络优化设置

\- \*\*DNS优化\*\*：改用 8.8.8.8 或 114.114.114.114

\- \*\*禁用IPv6\*\*：如果网络环境只使用IPv4

\- \*\*调整TCP参数\*\*：优化网络连接数



\## 安全设置建议

1\. \*\*用户账户控制(UAC)\*\*

&nbsp;  - 保持默认级别（不要完全关闭）

&nbsp;  - 防止恶意软件自动安装



2\. \*\*防火墙配置\*\*

&nbsp;  - 开启Windows Defender防火墙

&nbsp;  - 为常用程序添加入站规则



3\. \*\*更新策略\*\*

&nbsp;  - 开启自动更新

&nbsp;  - 定期检查驱动更新



\## 实用工具推荐

\- \*\*CCleaner\*\*：系统清理

\- \*\*Geek Uninstaller\*\*：彻底卸载软件

\- \*\*Everything\*\*：文件快速搜索

\- \*\*7-Zip\*\*：压缩解压工具



\## 故障排查

\### 常见问题解决

1\. \*\*系统卡顿\*\*

&nbsp;  - 检查资源占用（任务管理器）

&nbsp;  - 运行 `sfc /scannow` 修复系统文件

&nbsp;  - 更新显卡驱动



2\. \*\*网络问题\*\*

&nbsp;  - 重置网络：`netsh winsock reset`

&nbsp;  - 刷新DNS：`ipconfig /flushdns`



3\. \*\*蓝屏处理\*\*

&nbsp;  - 记录错误代码

&nbsp;  - 检查最近安装的软件/驱动

&nbsp;  - 使用Windows内存诊断工具
