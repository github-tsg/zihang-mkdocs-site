### **故障三：键盘部分按键失灵**  
**问题描述**  
键盘上部分按键输入无响应，或出现重复输入、乱码。

<img src="../QQ_1768881303315.png" width="30%" alt="图片比例30%">

**解决方案**  
1. **重启电脑**：可能是系统临时故障，重启后测试是否恢复。  
2. **检查键盘锁**：部分机型有键盘锁功能（如Fn+Win键），尝试按下解锁。  
3. **更新键盘驱动**：  
  打开“设备管理器” → “键盘” → 右键点击驱动 → 选择“更新驱动程序”。  
4. **外接键盘测试**：连接USB键盘，若外接键盘正常，则可能是笔记本键盘硬件故障。  
5. **清理键盘缝隙**：用软毛刷或吹风机冷风清理按键下方灰尘（关机状态下操作）。 
6. **联系客服支持**：若仍失灵，可能需要更换键盘模块。
